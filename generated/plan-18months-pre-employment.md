# Plan 18 Meses: Alcanzar Zona de Valor

## PHASE 1: BUILDING (0-6 meses | Mayo - Octubre 2026)

### Objetivo
Demostrar capacidad end-to-end en IA: desde concepto hasta producción con resultados medibles.

### Proyectos (YA EN MARCHA ✅)
**Support Agent AI** (Semana 1)
- Stack: Claude API + n8n + Supabase
- Outcome: MVP en producción
- Validar: Integration de LLM + automation + measurable impact

**RAG Document System** (Semanas 2-3)
- Stack: Claude embeddings + pgvector + Python + LangChain
- Outcome: Sistema de búsqueda semántica en producción
- Validar: LLM embeddings, vector search, performance

### Deliverables Esperados
- 2 proyectos en GitHub (portfolio visible)
- 2 proyectos demostrados en producción (tangible)
- 2 README exhaustivos (documentación clear)
- 1 video corto mostrando cómo funciona cada proyecto

### Qué Validamos
✅ Pragmatismo: "Hice, entrené, validé rápido"
✅ Ejecución: Projects shipped on time
✅ Technical competence: LLMs + APIs + automation + databases
✅ Production-ready thinking: No es demo, es sistema que funciona

### Success Metrics
- [ ] Support Agent: MVP listo semana 1
- [ ] RAG System: MVP listo semana 3

- [ ] Puedas explicar en 2 minutos por qué hiciste cada decisión

## PHASE 2: SPECIALIZATION (6-12 meses | Octubre 2026 - Abril 2027)

### Objetivo
Profundizar expertise en AI Implementation: ROI measurement, impact metrics, orchestration patterns.

### Skills a Profundizar

**1. ROI Measurement & Impact Metrics**
- Cómo medir el impacto real de IA en procesos
- Cost-benefit analysis
- Conocimiento de KPIs por industria
- Documentar aprendizajes de tus 2 proyectos: ¿Cuál fue el impacto? ¿Cuánto ahorro/valor?

**2. Stakeholder & Client Management**
- Entender necesidades del cliente (no solo técnicas)
- Saber decir "no" cuando IA no es la solución
- Traducir technical a business language
- Documentar: "En [proyecto], el cliente necesitaba [X], propuse [Y], resultado [Z]"

**3. n8n + API Patterns**
- Profundizar en n8n workflows (ya tienes base)
- Orquestación compleja: múltiples LLMs, fallover, logging
- Pattern library: "Cómo conectar X con Y de forma robusta"

**4. LLM Orchestration in Production**
- Prompt optimization para casos reales
- Token management
- Error handling
- Rate limiting
- Cost optimization

### Content & Documentation

**Case Studies (4-5 documentados)**

3. Experimento 1: [A definir durante Phase 1]
4. Experimento 2: [A definir durante Phase 1]

## PHASE 3: POSITIONING (12-18 meses | Abril - Noviembre 2027)

### Objetivo
Posicionarte como "AI Implementation Expert" + conseguir rol/oportunidades altos.

### Estrategia de Búsqueda

**Target Companies**
- Startups Series A-B que escalan con IA (producto + operaciones)
- Enterprises en post-PoC buscando implementación a escala
- Vendor-side: companies vendiendo soluciones IA a clientes

**Target Roles**
- AI Engineer (con specialization en implementation)
- AI Implementation Specialist (rol nuevo, menos competencia)
- AI Solutions Engineer (si vendor-side)
- Senior AI Engineer (si scaled bien)

**Target Geographies**
- Remote USA-based (salarios $160K-$250K)
- O Latinoamérica con remote multiplier (competencia menor, salarios ajustados)

### Network Strategy

**Targets de Network**
- AI engineers que IMPLEMENTAN (no researchers)
- Technical recruiters especializados en IA
- Founders/CTOs escalando con IA
- Ex-colleagues en Rappi (network base)

**Acciones**
- LinkedIn: Active engagement en posts de IA/implementation
- Twitter/X: Compartir learnings de implementación
- Communities: AI Discord servers, forums (Dev.to, etc)
- Talvez: Podcast/interviews sobre "pragmatic AI implementation"

### Elevator Pitch
"Soy AI Implementation Specialist. Ayudo companies a llevar IA a producción de forma pragmática, midiendo impacto real. Construí [X], [Y], [Z] en producción. Enfocado en ROI, no en experimentos."

### Success Metrics

## AJUSTES CUANDO CONSIGAS ROL

Una vez que consigas trabajo, este plan cambiará:

### Si consigues rol de **IC Engineer en startup/enterprise**:
- Phase 1 se completa con proyecto de la empresa
- Phase 2 se pausa (enfocarse en rol)
- Phase 3: Network/content se mantiene pero más low-key

### Si consigues rol de **AI Implementation Specialist**:
- Phase 1 se completa naturalmente (es tu job)
- Phase 2 se acelera (aprendes en el rol)
- Phase 3: Rápidamente posicionado para siguiente rol/level

### Si consigues rol de **Engineering Lead**:
- Recalibrar si realmente quieres liderazgo o prefieres IC
- Mantener skills técnicas
- Content pivot a "leadership + IA" en vez de pure implementation
